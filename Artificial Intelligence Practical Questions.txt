LIST OF PRACTICAL QUESTIONS:

1. Write a Prolog program to calculate the sum of two numbers.

2. Write a Prolog program to find the maximum of two numbers.

3. Write a Prolog program to find the factorial of an integer number.

4. Write a Prolog program to calculate Nth Fibonacci number.

5. Write a Prolog program to remove the Nth item from a list.

6. Write a Prolog program that checks if a list is a palindrome.

7. Write a Prolog program to implement max(X, Y, Max) so that Max is the maximum of two numbers X and Y.

8. Write a Prolog program to implement GCD of two numbers.

9. Write a Prolog program to implement maxlist(List,Max) so that Max is the greatest number in the list of numbers List using cut predicate.

10. Write a Prolog program to implement sumlist(List,Sum) so that Sum is the sum of a given list of numbers List.
